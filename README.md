# WhirlingBits-App

WhirlingBits-App is a comprehensive mobile solution designed for seamless IoT device management. It specializes in provisioning Espressif-based hardware and integrating it directly with the ThingsBoard platform, allowing for efficient device onboarding, configuration, and real-time monitoring.

## Features

*   **Espressif Device Provisioning:** Streamlined onboarding process for Espressif-based hardware (ESP32, ESP8266) via Bluetooth LE or Wi-Fi SoftAP.
*   **ThingsBoard Integration:** Seamless connection to the ThingsBoard IoT platform for centralized device management.
*   **Automated Onboarding:** Simplifies the registration of new devices to your IoT network.
*   **Remote Configuration:** Configure device parameters and settings directly from the mobile app.
*   **Real-Time Monitoring:** Visualize live telemetry data and device status updates instantly.
*   **Cross-Platform Support:** Built with Flutter for a consistent experience on Android, Web and Desktop.

## Installation

Currently, installation is only possible manually. You can build the app from source or install the APK directly if available.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/matti122/WhirlingBits-App.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd WhirlingBits-App
    ```
3.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
4.  **Run the app:**
    ```bash
    flutter run
    ```

Alternatively, check the [Releases](https://github.com/matti122/mealique/releases) page for the latest APK to install on your Android device.
